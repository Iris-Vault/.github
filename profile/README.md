## Welcome to Iris Vault

Privacy-first desktop application for local AI-powered screen capture and intelligence.

---

## What We're Building

A Tauri desktop application that provides AI-powered screen analysis while keeping all data completely local.

**Key Features**
- All processing happens locally, nothing leaves your device
- AI-powered screen capture and content analysis
- Cross-platform support for Windows, macOS, and Linux
- No internet connection required for core features

## Technology Stack

- **Framework**: Tauri (Rust + Web Technologies)
- **Processing**: Local AI models, on-device computation
- **Platforms**: Windows, macOS, Linux
